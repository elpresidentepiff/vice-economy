# Vice Economy Ops Dashboard

Static read-only dashboard for live economy state.

Open `index.html` in a browser, paste the Supabase URL and publishable key, then connect. The dashboard reads public RLS-protected endpoints only:

- `districts`
- `market_items`
- `crypto_exchange_rates`
- `police_incidents`
- `agents`

No service-role key belongs in this dashboard.
